# Company Portfolio Website

A modern and responsive **Company Portfolio Website** developed using **Core PHP**, **MySQL**, HTML5, CSS3, Bootstrap, and JavaScript. The project is designed to showcase a company's profile, services, projects, team members, blogs, and contact information through an easy-to-manage admin panel.

---

## 🚀 Features

### Frontend
- Responsive Home Page
- About Company
- Services
- Portfolio / Projects
- Team Members
- Testimonials
- Blog / News
- Contact Form
- Career / Job Listings
- Gallery
- Dynamic SEO Meta Tags
- Social Media Integration
- Google Maps Integration

### Admin Panel
- Secure Admin Login
- Dashboard
- Manage Company Information
- Manage Services
- Manage Projects
- Manage Team Members
- Manage Testimonials
- Manage Blogs
- Manage Gallery
- Manage Career Posts
- Contact Message Management
- Website Settings
- SEO Management
- File/Image Upload
- User Profile Management

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/company-portfolio.git
```

### 2. Move Project

Copy the project folder into your web server directory.

**XAMPP**

```
htdocs/company-portfolio
```

**Laragon**

```
www/company-portfolio
```

---

### 3. Create Database

Create a new MySQL database.

Example:

```
company_portfolio
```

---

### 4. Import Database

Import the SQL file into your database.

```
database/company_portfolio.sql
```

---

### 5. Configure Database

Edit:

```
config/database.php
```

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "company_portfolio";
```

---

### 6. Run Project

```
http://localhost/company-portfolio/
```

Admin Panel

```
http://localhost/company-portfolio/admin
```
